# Prácticas Async / Fetch API

Este repositorio contiene cuatro ejercicios hechos con **HTML, CSS y JavaScript**, usando **fetch** y **async/await** para consumir distintas APIs públicas.

## Ejercicios

1. **Generador de Perfiles Aleatorios**  
   - Archivo: `ejercicio1_async.html`  
   - API: [randomuser.me](https://randomuser.me)

2. **Galería Rick and Morty**  
   - Archivo: `ejercicio2_async.html`  
   - API: [rickandmortyapi.com](https://rickandmortyapi.com)

3. **Galería de Gatitos**  
   - Archivo: `ejercicio3_async.html`  
   - API: [thecatapi.com](https://thecatapi.com)

4. **Buscador de Libros**  
   - Archivo: `ejercicio4_async.html`  
   - API: [openlibrary.org](https://openlibrary.org)

## Cómo usar
- Abre el archivo **index.html** en tu navegador, o publícalo con **GitHub Pages**.  
- Desde ahí podrás acceder a cada ejercicio.

---

Hecho con JavaScript y APIs públicas.
